# Spaylater Manager

A lightweight desktop application for tracking shared "spaylater" (buy-now-pay-later)
items among a group. Built with Python + Tkinter. No database — just CSV + JSON files.

## Features

- **User management** — add/remove users with optional email
- **Item tracking** — assign items + prices to users per month
- **Monthly CSV reports** — auto-generated, human-editable, with backup
- **Snapshot export** — per-user CSV snapshots, saveable anywhere
- **Email delivery** — send snapshots via SMTP (Gmail App Password compatible)
- **Persistent state** — JSON state auto-saves; restores on relaunch
- **Offline-first** — runs with zero internet connection
- **Packagable** — builds to a single Windows EXE folder via PyInstaller

## Setup & Run

### 1. Clone / download the project

```bash
git clone https://github.com/yourname/spaylater.git
cd spaylater
```

### 2. Create a virtual environment

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

> **Note:** `tkinter` is included in standard Python. If you're on Linux and it's
> missing, install it with: `sudo apt install python3-tk`

### 4. Generate the icon (one-time)

```bash
python make_icon.py
```

### 5. Run the application

```bash
python main.py
```

## Gmail App Password Setup

To send emails, use a Gmail **App Password** (not your regular password):

1. Go to [myaccount.google.com](https://myaccount.google.com)
2. **Security** → **2-Step Verification** → enable it
3. **Security** → **App Passwords**
4. Create a password for **Mail** / **Windows Computer**
5. Copy the 16-character code
6. In Spaylater → **Settings**, enter:
   - SMTP Host: `smtp.gmail.com`
   - Port: `587`
   - Sender Email: `your@gmail.com`
   - App Password: *(paste 16-char code)*
   - TLS: ✓ checked
7. Click **Send Test Email** to verify

## Package as Windows EXE

```bash
# Make sure venv is active and pyinstaller is installed
pip install pyinstaller

# Build (uses main.spec)
pyinstaller main.spec

# Output:
#   dist/SpaylaterManager/SpaylaterManager.exe
#   dist/SpaylaterManager/ (all supporting files)
```

Distribute the entire `dist/SpaylaterManager/` folder.
The `data/` and `assets/` folders are bundled automatically.

> **Tip:** For a single-file EXE (slower startup), change `EXE(... exclude_binaries=True)`
> to `exclude_binaries=False` and remove the `COLLECT()` call in `main.spec`.

## Web Migration Notes

The codebase is structured for easy web migration:

| Layer | Current | Web equivalent |
|-------|---------|----------------|
| `StateManager` | JSON file | SQLite / PostgreSQL ORM |
| `CSVManager` | Local files | S3 / filesystem API |
| `ReportEngine` | Pure Python | Same (no changes needed) |
| `EmailSender` | smtplib | SendGrid / Resend API |
| UI panels | tkinter | FastAPI + React components |

Each layer has no cross-dependencies — swap one without touching the others.
